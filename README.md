# 📸 Gather: Event Memory Collector

**Transform scattered photos and videos into one shared event timeline.**

Gather allows everyone at an event—weddings, parties, trips—to upload their memories into a single, collaborative album in real-time.

---

## Core Features
- **Instant Access:** Join via QR Code, Event Code, or Link.
- **Shared Gallery:** All photos and videos in one organized timeline.
- **Micro-Social:** Like, comment, and react to memories.
- **Live Slideshow:** Auto-generated presentation for event screens.
- **Privacy Controls:** Host-managed upload and download permissions.

## Tech Stack
- **Frontend:** React + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Real-time:** Socket.io
- **Media:** Cloudinary / Firebase Storage

## 🚀 Getting Started
1. `npm install`
2. `npm run dev`
