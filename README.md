# Talk With EMO

> An emotional intelligence platform where you talk to the emotion you're feeling — not about it.

**Live →** [mnsbaanu.github.io/TalkWith_EMO](https://mnsbaanu.github.io/TalkWith_EMO/)

---

## Overview

Talk With EMO is a cinematic, scroll-driven web experience built around the five core emotions from Pixar's *Inside Out* — Joy, Sadness, Anger, Fear, and Disgust. Each emotion is a fully realized character with its own personality, tone, color palette, and conversational style.

Users scroll through immersive parallax sections, discover which emotion resonates with them, and open a real-time chat with that EMO.

---

## Features

- Cinematic parallax scrolling powered by GSAP + Lenis
- Five individual emotion sections with character art and scroll animations
- In-browser chat with keyword-matched, personality-driven responses
- 3-question emotion quiz to help users identify what they're feeling
- Fully responsive layout with split hero, character imagery, and animated transitions
- Deployed via GitHub Actions to GitHub Pages

---

## Tech Stack

React 19 · TypeScript · Vite · Tailwind CSS v4 · GSAP · Lenis · Sora · DM Sans

## Getting Started

```bash
git clone https://github.com/MNSBaanu/TalkWith_EMO.git
cd TalkWith_EMO
npm install
npm run dev
```

```bash
# Production build
npm run build

# Deploy to GitHub Pages
npm run deploy
```
